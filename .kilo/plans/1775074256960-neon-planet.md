### FASE 1: Motor de Ubicación y Compatibilidad (Semana 1-2)

#### Tarea 1.1: Completar integración de compatibilidad en LocationEngine
**Archivo:** `backend/services/location_engine.py`  
**Problema:** El método `_verificar_compatibilidad_vecinos` retorna siempre True  
**Solución:** Integrar llamada a `CompatibilidadService.verificar_compatibilidad()`  

```python
# Cambiar líneas 196-247 para usar el servicio de compatibilidad
from services.compatibilidad import CompatibilidadService

def _verificar_compatibilidad_vecinos(db, muestra, hilera):
    # Obtener muestras vecinas
    vecinos = CompatibilidadService.get_vecinos(db, hilera.id)
    
    if not vecinos:
        return {"compatible": True, "mensaje": "Sin vecinos"}
    
    # Verificar cada vecino
    clase_muestra = muestra.clase_peligro.codigo if muestra.clase_peligro else None
    if not clase_muestra:
        return {"compatible": True, "mensaje": "Sin clase de peligro"}
    
    for vecino in vecinos:
        resultado = CompatibilidadService.verificar_compatibilidad(
            clase_muestra, vecino["clase_peligro"]
        )
        if not resultado["compatible"]:
            return {
                "compatible": False,
                "mensaje": resultado["mensaje"],
                "incompatible_con": vecino["nombre"]
            }
    
    return {"compatible": True, "mensaje": "Compatible con todos los vecinos"}
```
